# Oustagram v1.0

## This is the final project for GA React development online course.

### Only use React Hook, no Class compoents.

## Deployment

- Firebase Hosting
- Firebase Database
- Firebase Functional API
- Firebase Storage

## Dependencies

- Material-UI
- Axios
- React-Router-Dom V6
- Redux
- Redux-Thunk

## Installation

> yarn install

## Execute

> yarn start
